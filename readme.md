# Catch the train Server

## Setup Dev

### Install

> npm i

### Start

> npm start

### Test

Open a browser at ['http://localhost:3034/schedules'](http://localhost:3034/schedules) and check the result:

```json
{
  "nextDeparture": {
    "time": "2020-03-09T18:43:52+00:00"
  }
}
```
